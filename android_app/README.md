# UDX Android App

This is the Android client for UDX.

## Prerequisites
- Android Studio Iguana or newer
- Android SDK 34
- JDK 17

## How to Run

1.  Open **Android Studio**.
2.  Select **Open**.
3.  Navigate to `android_app` folder (the one containing `build.gradle.kts` and `settings.gradle.kts`).
4.  Click **OK**.
5.  Wait for Gradle sync to complete.
6.  Select an emulator or connected device.
7.  Click the **Run** (Play) button.

## Structure
- `app/src/main/java/com/udx/app`: Kotlin Source Code
- `app/build.gradle.kts`: App module configuration
